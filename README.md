## 📌 Project Overview

This project demonstrates a complete **end-to-end data analytics workflow** using customer shopping behavior data.  
The objective of this project is to analyze customer purchasing patterns, identify business insights, and create an interactive dashboard for decision-making.

The project simulates how real-world data analysts work with raw business data to generate actionable insights using:

- 🐍 Python for Data Cleaning & EDA
- 🗄️ SQL for Business Analysis
- 📊 Power BI for Dashboard Visualization

# 🚀 Business Problem

Retail businesses generate massive amounts of customer transaction data every day.  
The challenge is converting this raw data into meaningful insights that can help businesses:

- Understand customer behavior
- Improve customer retention
- Increase revenue
- Identify top-performing products
- Optimize marketing strategies

This project focuses on solving these problems using data analytics techniques.

# 📂 Project Workflow

## 1️⃣ Data Cleaning & Preparation
Performed using Python:
- Removed duplicates
- Handled missing values
- Standardized data types
- Cleaned inconsistent records

---

## 2️⃣ Exploratory Data Analysis (EDA)

Analyzed:
- Customer demographics
- Spending behavior
- Product category trends
- Purchase frequency
- Revenue contribution

---

## 3️⃣ SQL Business Analysis

Created SQL queries to answer business questions such as:

- Which customer segment generates the highest revenue?
- Which products are purchased most frequently?
- What is the average customer spending?
- Which age group contributes the most sales?
- How does customer loyalty impact revenue?

---

## 4️⃣ Power BI Dashboard

Built an interactive dashboard including:

✅ KPI Cards  
✅ Revenue Trends  
✅ Customer Segmentation  
✅ Product Performance  
✅ Interactive Filters & Slicers  
✅ Purchase Behavior Analysis

# 📊 Key Insights

📌 High-value customers contribute significantly to total revenue.

📌 Certain product categories consistently outperform others.

📌 Repeat customers show stronger purchasing behavior compared to new customers.

📌 Customer demographics play a major role in shopping trends.
